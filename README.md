# 色彩消除大挑战 (Color Crush Challenge)

一款休闲益智类游戏，玩家通过匹配相同颜色的方块来消除并获得分数。

## 游戏特色

- 🎯 简单易学，老少皆宜
- 🌈 丰富多彩的游戏体验
- 🏆 全球排行榜竞争
- 👥 好友挑战模式
- 🎮 100+精心设计的关卡

## 游戏玩法

1. 通过交换相邻方块，形成3个或以上相同颜色的连续排列来消除
2. 消除方块获得分数，达成关卡目标即可过关
3. 随着关卡推进，引入特殊方块和障碍物增加挑战

### 特殊方块

- 🚀 火箭方块: 消除一整行或列的所有方块
- 💣 炸弹方块: 消除周围3x3区域的所有方块
- 🌈 彩虹球: 可以与任何颜色匹配

## 技术特性

- 使用Godot游戏引擎开发
- 支持跨平台发布 (iOS, Android, Web)
- 内置广告系统和内购功能
- 社交排行榜系统

## 开发

### 本地运行

1. 安装Godot 3.5+
2. 下载项目代码
3. 在Godot编辑器中打开`project.godot`文件

### 构建Web版本

项目配置了GitHub Actions自动构建Web版本，构建产物会自动部署到GitHub Pages。

## 商业化

游戏采用免费增值模式，通过广告和内购实现盈利。
